一直想着更新博客，但是又不知道更新什么，突然想起博客部署在 GitHub 上，上面的文章很容易被搜索引擎搜索到。我是那种不太敢展现自我的人，虽然一般也没人搜索我的文章（应该），但是我自搜时看到就感觉很害羞😳，为了避免被必应搜索到，我开始想迁移博客，我个人是觉得静态博客是比动态博客要好，因为静态博客加载速度快:spoiler[因为可以白嫖 GitHub Page 而不用买服务器]，而且我也没有自己的域名和云服务器，于是我就把博客放到了我本地的小笔记本电脑上了，然后通过樱花内网穿透上传到公网，再在 GitHub 上用 iframe 标签进行展示，这样一来我的博客还是静态的，也不会被必应搜索到，后续想更换博客也不用动到 GitHub 仓库。
